CUADRÍCULA

La cuadrícula de Utuqu está construida en base a 12 columnas que ocupan espacios equivalentes en un lienzo de 1200px de anchura y es mobile-first, lo que quiere decir que los estilos se heredan "hacia arriba".

Usa la clase .bloque para crear una division horizontal y anida tantas .peque-* como columnas necesites. Agrega .medio-* o .grande-* para definir el tamaño en pantallas de mayores dimensiones.

Para conseguir columnas sin espacio de separación entre ellas, agrega .sin-medianil al elemento .bloque.

Cuadrícula automática

Para crear rejillas automáticas de anchuras equivalentes podemos usar la clase .bloque-auto dentro de un elemento .bloque. Esta rejilla escala desde cuatro columnas en pantallas grandes, a dos columnas en las pantallas más pequeñas.

Lista de clases

A continuación se detallan todas las clases que tienen relación con la cuadrícula.

Clase Rango
.peque-[1-12] Entre 0 y 599px Genera divisiones verticales en tamaños de pantalla pequeños.
.medio-[1-12] Entre 600 y 1199px Genera divisiones verticales en tamaños de pantalla medianos.
.grande-[1-12] De 1200px en adelante Genera divisiones verticales en tamaños de pantalla grandes.
.sin-medianil todos Elimina los espacios entre divisiones verticales.
.bloque-auto todos Crea una rejilla de al menos cuatro elementos que se adapta al tamaño de la ventana.
.alturas-iguales De 600px en adelante Aplica comportamiento de tabla al elemento.
.iguala De 600px en adelante Aplica comportamiento de celda de tabla al elemento. En pantallas pequeñas los elementos se apilarán.
.columna-centrada todos Centra una división vertical que sea menor de 12 columnas.

IMÁGENES

Las imágenes son responsivas por defecto. Basta con añadir un elemento img y automáticamente se redimensionará cuando cambie el tamaño de la pantalla. Para conseguir imágenes redondas añade la clase .imagen-redondeada a una imagen con la misma altura y anchura.

Muestra

  <img src="https://unsplash.it/1920x700" alt="imagen">
  <img class="imagen-circular" src="https://unsplash.it/600x600" alt="imagen">

IFRAMES

Combinando las clases .iframe y .ratio--* se consiguen iframes responsivos en los dos ratios más habituales. También puedes crear tu propia clase .ratio--* para adaptarte a formatos menos convencionales.

Muestra

  <figure class="iframe ratio--4-3">
    <iframe
      src="https://www.youtube.com/embed/SbCt-iXIXlQ?rel=0&amp;controls=0&amp;showinfo=0"
      frameborder="0"
      webkitallowfullscreen
      mozallowfullscreen
      allowfullscreen>
    </iframe>
  </figure>

BOTONES

Los botones se construyen usando el elemento button pero en algunas situaciones es necesario (o inevitable) usar anclas para este fin, o incluso otro tipo de elemento. La clase .boton es suficientemente flexible para aplicarse a cualquier tipo de elemento. La clase se extiende con .secundario, .plano y .redondo.

Muestra

  <button class="boton">BOTÓN NORMAL</button>

ACORDEÓN

El acordeón de Utuqu está implementado mediante la etiqueta nativa <details> a la que tan solo se le han añadido algunos estilos, y su proposito es proporcionar una manera de mostrar contenido secundario. Para marcar un elemento como abierto solo debemos añadir el atributo open.

NOTA: <details> está soportado en todos los navegadores excepto (¡sorpresa!) en IE y Edge. Datos de soporte actualizados en caniuse.com.

DESPLEGABLE

Este componente genera un menú desplegable básico y puede ser lanzado con un a y con elementos de formulario como input y button. Como se utiliza la propiedad :focus para emular el estado, para usarlo en otro tipo de elementos hay que añadir tabindex="-1" al elemento que aloja la clase .desplegable_abre. La clase .boton solo se usa con fines estilísticos.
